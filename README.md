# business-decision-analysis
Data science project for business expansion planning by analyzing regional potential, customer distribution, and market trends to support strategic branch location decisions.

## Objective
This project aims to identify the most strategic location for opening a new business branch using data analysis and business intelligence approaches.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Analysis Performed
- Regional comparison
- Customer distribution analysis
- Sales trend visualization
- Business opportunity evaluation

## Results
The analysis provides data-driven recommendations for selecting potential business branch locations.
